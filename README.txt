# Client Side Block Overlay 1.20

## Build instructions
1. Open `ClientSideBlockOverlay.csproj` in Visual Studio 2022.
2. Make sure the paths to VintagestoryAPI.dll and VintagestoryLib.dll are correct.
3. Build the project (Ctrl + Shift + B).
4. Copy `ClientSideBlockOverlay.dll` and `modinfo.json` to:
   `D:\Users\gLiczu\AppData\Roaming\Vintagestory\Mods\ClientSideBlockOverlay\`
5. Launch the game and enable the mod.

## Features
- Fully client-side overlay.
- Toggle ON/OFF with `[` key.
- Customizable highlight blocks via `ModConfig.json`.
- Adjustable search distance.
